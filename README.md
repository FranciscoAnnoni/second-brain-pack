# Second Brain Pack

A minimal, installable second brain powered by AI. Works with **Gemini CLI** (free) or **Claude Code**.

You get a personal vault of Markdown files and a daily brief that summarizes your tasks, last session, and what needs attention — generated by the AI agent of your choice.

---

## What you get after installing

- A local vault (Markdown files, no cloud lock-in)
- A personalized AI assistant with your name and style
- A `brief` command that gives you your daily context in seconds
- A desktop shortcut (`Aria.app` on Mac, `Aria.bat` on Windows) to launch it with a double-click
- Git history of everything from day one

---

## Requirements

- Mac or Windows
- Internet connection (for downloads)
- Python 3.8 or later ([python.org](https://www.python.org/downloads/))
- Git ([git-scm.com](https://git-scm.com/downloads))

The installer handles everything else (Gemini CLI, Claude Code, MCP tools).

## Install

### Mac

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/FranciscoAnnoni/second-brain-pack/main/bootstrap/mac.sh)"
```

Or if you prefer to clone first:

```bash
git clone https://github.com/FranciscoAnnoni/second-brain-pack
cd second-brain-pack
bash bootstrap/mac.sh
```

### Windows

Open PowerShell and run:

```powershell
iwr -useb https://raw.githubusercontent.com/FranciscoAnnoni/second-brain-pack/main/bootstrap/windows.ps1 | iex
```

Or clone first:

```powershell
git clone https://github.com/FranciscoAnnoni/second-brain-pack
cd second-brain-pack
powershell -ExecutionPolicy Bypass -File bootstrap\windows.ps1
```

The installer opens a window and walks you through the setup. Takes about 5 minutes.

## How it works

```
vault/
├── SOUL.md          ← who your AI assistant is
├── MEMORY.md        ← long-term memory index
├── pendientes.md    ← open tasks (source of truth)
├── daily/           ← append-only session logs
├── config/
│   └── agent.yaml   ← agent config (auto-generated)
└── brief.py         ← the brief script
```

`brief.py` reads `pendientes.md` and your latest daily log, builds a prompt, and calls the configured agent. The output is your daily brief.

---

## Customization

**Change the assistant's personality** — edit `SOUL.md` in your vault. The compact version (`~500 tokens`) works best with Gemini free. The standard version (`~1500 tokens`) gives Claude more context.

**Add tasks** — edit `pendientes.md` directly or ask your assistant to add them.

**Switch agents** — edit `config/agent.yaml` and change the `agent` field to `claude`, `gemini`, or `other`.

## Security note

**Mac:** The desktop `.app` is created locally by the installer (not downloaded), so it does not trigger Gatekeeper. If you see a security warning the first time, right-click → Open.

**Windows:** The `.bat` shortcut on your desktop is a plain text file. Windows may show a SmartScreen warning — click "More info" → "Run anyway".
